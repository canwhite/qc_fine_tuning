# 蒸馏细节问题总结

> 本文档总结蒸馏过程中的关键技术细节和数据结构变化。

---

## 一、关键概念详解

### 1. 什么是 log_softmax？

```
log_softmax(x) = log(softmax(x))
               = log(e^xi / sum(e^xj))
               = xi - log(sum(e^xj))
```

**为什么用 log_softmax 而不是 log(softmax)？**
- 数值稳定性：直接计算 log_softmax 避免 softmax 输出很小值时 log 的数值问题
- PyTorch 内部优化过，更高效

---

### 2. KL 散度是什么？

```
KL(P, Q) = Σ P(x) × log(P(x) / Q(x))
         = Σ P(x) × (log P(x) - log Q(x))
```

**直观理解**：衡量两个概率分布的"距离"
- P = 教师分布，Q = 学生分布
- KL = 0 表示完全相同
- KL 越大表示差距越大

---

### 3. 为什么乘 T²？

```
softmax(x/T) 的梯度 ≈ (1/T) × softmax(x) 的梯度

当 T = 2 时，梯度缩小 2 倍
乘 T² = 4 可以补偿回来，保持梯度量级一致
```

---

### 4. 交叉熵用的是 -log 吗？

```
CrossEntropy = -log(P(正确答案))

例子：
真实标签 = "猫"
预测概率 = [0.64, 0.24, 0.10, 0.02, 0.01]
            猫    狗    鸟    鱼    兔

交叉熵 = -log(0.64) = 0.45
```

**是的！交叉熵 = -log(正确答案的概率)**

---

### 5. α 是什么？从哪来？

```
α = 软标签权重（你自己设定的超参数）

total_loss = α × soft_loss + (1-α) × hard_loss
           = 0.7 × soft_loss + 0.3 × hard_loss

常见取值：0.5 ~ 0.9
推荐值：0.7
```

## 二、蒸馏的数据结构变化

```
┌─────────────────────────────────────────────────────────────────────────┐
│                           输入数据                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  training_texts: List[str]                                              │
│  [                                                                       │
│    "人工智能是计算机科学的一个分支...",                                   │
│    "机器学习是人工智能的核心...",                                        │
│    ...                                                                   │
│  ]                                                                       │
│                                                                          │
│  形状: [batch_size] = [5] 条文本                                         │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
                                    ↓ tokenizer
┌─────────────────────────────────────────────────────────────────────────┐
│                        Tokenized 数据                                    │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  input_ids: tensor                                                       │
│  形状: [batch_size, seq_len] = [5, 64]                                   │
│                                                                          │
│  [                                                                       │
│    [101, 102, 103, 104, ..., 0, 0, 0],  ← 第1条文本的 token IDs          │
│    [101, 205, 306, 407, ..., 0, 0, 0],  ← 第2条文本的 token IDs          │
│    ...                                                                   │
│  ]                                                                       │
│                                                                          │
│  attention_mask: tensor                                                  │
│  形状: [batch_size, seq_len] = [5, 64]                                   │
│                                                                          │
│  [                                                                       │
│    [1, 1, 1, 1, ..., 0, 0, 0],  ← 1=有效token, 0=padding                │
│    ...                                                                   │
│  ]                                                                       │
│                                                                          │
│  labels: tensor (= input_ids，padding 位置设为 -100)                     │
│  形状: [batch_size, seq_len] = [5, 64]                                   │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
                                    ↓ 教师模型前向传播
┌─────────────────────────────────────────────────────────────────────────┐
│                        教师模型输出                                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  teacher_logits: tensor                                                  │
│  形状: [batch_size, seq_len, vocab_size] = [5, 64, 151936]               │
│                                                                          │
│  含义：每个位置的每个词的原始分数                                         │
│                                                                          │
│  [                                                                       │
│    [                          ← 第1条文本                                │
│      [2.1, -0.5, 1.3, ...],  ← 位置1，所有词的 logits                    │
│      [0.8, 1.2, -0.3, ...],  ← 位置2，所有词的 logits                    │
│      ...                                                                 │
│    ],                                                                    │
│    [                          ← 第2条文本                                │
│      ...                                                                 │
│    ],                                                                    │
│    ...                                                                   │
│  ]                                                                       │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
                                    ↓ 学生模型前向传播
┌─────────────────────────────────────────────────────────────────────────┐
│                        学生模型输出                                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  student_logits: tensor                                                  │
│  形状: [batch_size, seq_len, vocab_size] = [5, 64, 151936]               │
│                                                                          │
│  结构与 teacher_logits 相同                                              │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
                                    ↓ 计算蒸馏损失
┌─────────────────────────────────────────────────────────────────────────┐
│                        损失计算过程                                       │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  Step 1: 温度缩放                                                        │
│  ─────────────────                                                       │
│  teacher_logits / T                                                      │
│  形状: [5, 64, 151936] → [5, 64, 151936]（数值缩小）                     │
│                                                                          │
│  student_logits / T                                                      │
│  形状: [5, 64, 151936] → [5, 64, 151936]（数值缩小）                     │
│                                                                          │
│                                                                          │
│  Step 2: Softmax / Log_Softmax                                          │
│  ─────────────────────────────                                          │
│  soft_targets = softmax(teacher_logits/T, dim=-1)                        │
│  形状: [5, 64, 151936]                                                   │
│  含义：教师的概率分布（每个位置所有词概率和=1）                            │
│                                                                          │
│  soft_student = log_softmax(student_logits/T, dim=-1)                    │
│  形状: [5, 64, 151936]                                                   │
│  含义：学生的 log 概率分布                                               │
│                                                                          │
│                                                                          │
│  Step 3: KL 散度                                                         │
│  ───────────────                                                         │
│  kl_div(soft_student, soft_targets)                                      │
│  形状: [5, 64] → 标量                                                    │
│  含义：学生和教师分布的"距离"                                            │
│                                                                          │
│  soft_loss = KL × T²                                                     │
│  形状: 标量                                                              │
│                                                                          │
│                                                                          │
│  Step 4: 交叉熵（硬标签）                                                │
│  ───────────────────────                                                 │
│  cross_entropy(student_logits, labels)                                   │
│  形状: 标量                                                              │
│  含义：学生预测正确答案的能力                                            │
│                                                                          │
│                                                                          │
│  Step 5: 总损失                                                          │
│  ─────────────                                                           │
│  total_loss = α × soft_loss + (1-α) × hard_loss                          │
│  形状: 标量                                                              │
│                                                                          │
│  例如：total_loss = 0.7 × 2.3 + 0.3 × 1.5 = 2.06                         │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
                                    ↓ 反向传播
┌─────────────────────────────────────────────────────────────────────────┐
│                        梯度与更新                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  total_loss.backward()                                                   │
│  → 计算学生模型每个参数的梯度                                             │
│                                                                          │
│  梯度形状 = 参数形状                                                     │
│  例如：                                                                   │
│    - embedding.weight: [151936, 896]                                     │
│    - layer.0.weight: [896, 896]                                          │
│    - ...                                                                 │
│                                                                          │
│  optimizer.step()                                                        │
│  → 更新学生模型参数（教师模型不变！）                                     │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
```

## 三、数据形状汇总表

| 变量 | 形状 | 含义 |
|------|------|------|
| `training_texts` | `[batch_size]` | 原始文本列表 |
| `input_ids` | `[batch, seq_len]` | Token IDs |
| `attention_mask` | `[batch, seq_len]` | 有效位置标记 |
| `labels` | `[batch, seq_len]` | 真实标签 |
| `teacher_logits` | `[batch, seq_len, vocab_size]` | 教师原始输出 |
| `student_logits` | `[batch, seq_len, vocab_size]` | 学生原始输出 |
| `soft_targets` | `[batch, seq_len, vocab_size]` | 教师概率分布 |
| `soft_student` | `[batch, seq_len, vocab_size]` | 学生 log 概率 |
| `soft_loss` | 标量 | KL 散度损失 |
| `hard_loss` | 标量 | 交叉熵损失 |
| `total_loss` | 标量 | 总损失 |

---

## 四、一句话总结数据流

```
文本 [batch]
  → tokenizer
→ tokens [batch, seq_len]
  → 教师模型
→ logits [batch, seq_len, vocab]
  → softmax
→ 概率分布 [batch, seq_len, vocab]
  → KL散度
→ 标量损失
  → 反向传播
→ 更新学生模型
```

---

## 五、关键公式速查

| 概念 | 公式 |
|------|------|
| Softmax | `softmax(x) = e^xi / Σe^xj` |
| Log Softmax | `log_softmax(x) = xi - log(Σe^xj)` |
| 温度缩放 | `softmax(x/T)` |
| KL 散度 | `KL(P,Q) = Σ P × log(P/Q)` |
| 交叉熵 | `CE = -log(P_correct)` |
| 蒸馏损失 | `L = α × KL_loss + (1-α) × CE_loss` |
| 梯度补偿 | `KL_loss × T²` |
